# 📝 To-Do List CLI App

A simple command-line To-Do List application built using Python.

## 🚀 Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using JSON

## 🛠️ Technologies Used
- Python
- JSON
- File Handling

## ▶️ How to Run
1. Install Python
2. Run:
   python todo.py
